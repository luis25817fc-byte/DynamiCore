# DynamiCore Research Framework

Repositorio para el paper:

**Finite-Size Entropy Condensation and Computational Cost Singularity in Exponentially Scaling Discrete State Spaces**

## Estructura

paper/
- dynamicore.tex → manuscrito principal

src/
- generate_figures.py → generación de figuras

requirements.txt → dependencias

## Instalación

pip install -r requirements.txt

## Generación de figuras

python src/generate_figures.py

## Compilación LaTeX

cd paper
pdflatex dynamicore.tex
